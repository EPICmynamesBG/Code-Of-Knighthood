	In history, Knights are frequently defined by a code of rules known as the Code of Chivalry. What is the Code of Chivalry? A general definition would be that the Code of Chivalry is a set of societal laws that dictate a knights behavior. Similarly, in modern society, there is a set of societal laws that dictate how males and females interact, especially on the personal level. Thus, the purpose of this reading is to describe exactly how the Medieval Code of Chivalry never truly died; rather, the Code was simply adapted to fir the modern perspective.
